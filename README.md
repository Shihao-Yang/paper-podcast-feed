# Paper Podcast Feed

Personal RSS feed for AI-narrated readings of machine-learning, statistics, and epidemiology papers. Generated by the [paper_podcast](https://github.com/Shihao-Yang/paper-podcast) CLI.

- Feed URL: `https://shihao-yang.github.io/paper-podcast-feed/feed.xml`
- Episodes live under `episodes/`.

This repo is published via GitHub Pages from the `main` branch.
